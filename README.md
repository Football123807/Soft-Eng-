# Soft-Eng-
Software Engineering work - A Collection of Programs

## Overview
This repository contains a set of programs demonstrating various software engineering concepts and utilities across different programming languages.

## Programs

### Python Programs

#### 1. Calculator (`programs/python/calculator.py`)
A simple interactive calculator that performs basic arithmetic operations.

**Features:**
- Addition, subtraction, multiplication, and division
- Interactive command-line interface
- Error handling for invalid inputs and division by zero

**Usage:**
```bash
python3 programs/python/calculator.py
```

**Example:**
```
Enter calculation (e.g., 5 + 3): 10 + 5
Result: 15.0
```

#### 2. Text Analyzer (`programs/python/text_analyzer.py`)
Analyzes text input and provides detailed statistics.

**Features:**
- Counts characters, words, lines, sentences, and paragraphs
- Handles multi-line text input

**Usage:**
```bash
python3 programs/python/text_analyzer.py
```

### JavaScript Programs

#### 3. Data Formatter (`programs/javascript/data_formatter.js`)
Formats data in multiple formats: JSON, CSV, and table.

**Features:**
- JSON formatting with indentation
- CSV export with proper escaping
- ASCII table formatting

**Usage:**
```bash
node programs/javascript/data_formatter.js
```

#### 4. Number Utilities (`programs/javascript/number_utils.js`)
Provides various number-related utility functions.

**Features:**
- Prime number detection
- Factorial calculation
- Fibonacci sequence generation
- GCD (Greatest Common Divisor) and LCM (Least Common Multiple) calculation

**Usage:**
```bash
node programs/javascript/number_utils.js
```

### Java Programs

#### 5. String Utilities (`programs/java/StringUtils.java`)
Provides string manipulation utilities.

**Features:**
- String reversal
- Palindrome detection
- Vowel counting
- Word capitalization

**Usage:**
```bash
# Compile
javac programs/java/StringUtils.java

# Run
java -cp programs/java StringUtils
```

## Requirements

- **Python**: Python 3.6 or higher
- **JavaScript**: Node.js 12 or higher
- **Java**: JDK 8 or higher
